Blueprint3D React Migration Guide

Setup Instructions

  1. Navigate to the react-app directory: 
    cd react-app
  2. Install dependencies: 
    npm install
  3. Copy Blueprint3D build files: 
    # Copy the compiled Blueprint3D library
Copy-Item ..\dist\blueprint3d.js public\blueprint3d.js

# Copy assets (models, textures, etc.)
Copy-Item -Recurse ..\example\models public\models
Copy-Item -Recurse ..\example\rooms public\rooms
  4. Start the development server: 
    npm start

Project Structure

  • src/App.tsx - Main application component
  • src/hooks/useBlueprint3D.ts - Custom hook for Blueprint3D integration
  • src/components/FloorPlanner.tsx - 2D floor plan editor component
  • src/components/ThreeViewer.tsx - 3D viewer component
  • src/components/Toolbar.tsx - Top toolbar with view mode toggle
  • src/components/Sidebar.tsx - Item and texture selection sidebar
